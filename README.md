# Ash-Roomrent-Expense-calculate

Mobile-friendly app to track personal monthly expenses and split shared room rent with members.

## Features

- Personal expense calculator (salary vs expenses)
- Room rent split with members and shared bills
- Save history, WhatsApp/SMS sharing
- Saved room profiles for quick monthly calculation

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Live

Built with Vite, React, TypeScript, and Tailwind CSS.
